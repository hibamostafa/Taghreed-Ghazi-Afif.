# 🎯 تعليمات سريعة للرفع على Vercel

## الخطوة 1: تحضير المشروع ✅
```bash
cd flutterfolio-showcase-main
npm install  # تم! ✓
npm run build  # تم! ✓
```

## الخطوة 2: رفع على GitHub
```bash
git add .
git commit -m "Ready for Vercel - Website Portfolio with Firebase &  Web App "
git push
```

## الخطوة 3: الرفع على Vercel (الأسهل) 🚀

1. توجه إلى https://vercel.com
2. اضغط "Add New" → "Project"
3. اختر "Import Git Repository"
4. اختر مشروعك من GitHub
5. اضغط "Deploy"

**انتظر 2-3 دقائق وستحصل على رابط موقعك!**
